# Logo Assets

## Kenya Government Coat of Arms

The official Kenya Coat of Arms logo should be placed in this directory.

### Download Sources

**Recommended (Official):**
- **Wikimedia Commons**: https://commons.wikimedia.org/wiki/File:Coat_of_arms_of_Kenya_(Official).svg
  - Free to use with attribution
  - Available in SVG format (3.46 MB)
  - High quality vector format

**Alternative Sources:**
- **CreativeSoko**: https://creativesoko.com/downloads/kenya-coat-of-arms-logo/
  - Formats: PNG, PDF, EPS, AI, SVG
  - Official government logo

- **SeekLogo**: https://seeklogo.com/vector-logo/294798/kenya-coat-of-arms
  - Formats: PNG, SVG, AI, EPS, CDR
  - Free download

### File Naming Convention

- `kenya-coat-of-arms.svg` - Vector format (recommended)
- `kenya-coat-of-arms.png` - PNG format for compatibility

### Usage in the System

The logo will be used in:
- Login page header
- Dashboard header
- Generated reports
- Email notifications
